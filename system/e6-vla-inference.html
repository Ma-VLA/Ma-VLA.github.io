Single-owner boundary

Read and command channels have separate owners.

camera_state_node exclusively owns Dobot FeedBack :30005 and both HIK/ZED camera devices. executor_supervisor_node exclusively owns Dashboard :29999 for MovJ, ServoJ, and ToolDO commands. The policy server is outside ROS2, so its live inference latency appears in the server console rather than this node graph or the current MCAP record.

Compared with the Stage 2 collection system

The xArm6 pipeline separates human teleoperation and data collection across multiple packages and processes. The E6 pipeline runs observation, policy bridging, execution, and context as an autonomous loop within e6_vla_ros; there is no continuous human teleoperation point because the inference policy is the operator.
