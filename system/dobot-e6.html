Dobot Magician E6 + Suction

The current FlowBridge system adapts a π0.5 policy to a 6-DoF Dobot arm with a binary suction gripper and executes 16-step joint-space action chunks through a ROS2-connected runtime.

Validatedπ0.5 · v236-DoF · 7D action

16 × 7 action chunk

[Δj1, Δj2, Δj3, Δj4, Δj5, Δj6, suction_abs]

Model

π0 lineage, π0.5 implementation.

The publication lineage begins with Pi-Zero adaptation. The current representative checkpoint is pi05_e6_v23_lora at 20,000 steps.

ROS2 runtime

Inference to hardware, without MoveIt2.

The current VLA runtime sends validated joint-space targets directly through Dobot command modes. MoveIt2 is not part of this deployed path.
