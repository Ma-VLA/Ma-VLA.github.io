Overview

FlowBridge is a manipulation-focused VLA that reuses frozen vision-language priors and learns robot-specific action heads from demonstration data. The project aims to bridge semantic task understanding and executable real-world robot control.

Instead of showing an unfinished paper or embedded video, this version keeps the page focused on the project status, dataset availability, and open-source components. That makes the page cleaner for the current stage while leaving room to add a paper section later.

Pipeline

The pipeline consists of embodiment data collection, flow-matched action head training, and real-world policy deployment.

01. Data Collection

Teleoperation captures robot-camera trajectories, suction grasp interactions, calibration data, and replayable manipulation episodes.

02. Flow-Matched Action Head

Training reuses frozen vision-language priors while learning a robot-specific action head that adapts semantic understanding to control.

03. Real-World Control Transfer

The learned policy is deployed to the physical Dobot E6 setup for real-world manipulation execution across scenes.
