Project 01 · Dobot E6 evaluation

Task success and control performance.

All values on this page describe the validated Dobot Magician E6 setup. Success rates include the full number of trials, and control measurements are reported as mean ± standard deviation.

Evaluation set

Four colors and four object categories.

Each color and each object category was evaluated in 20 trials, producing 80 trials per recognition task.

Control comparison

ServoJ completed tasks faster in this setup.

The comparison is specific to the reported E6 evaluation conditions and should not be interpreted as a universal ranking of command modes.

Scope and interpretation

What these results do—and do not—show.

  • The numbers apply to the validated Dobot E6 setup.
  • The current results do not establish cross-robot generalization.
  • Exact control-mode metadata for each public demo video remains under verification.
